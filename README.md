# 🏭 302_sector_analyzer — 週刊セクター分析

東証17業種 ETF（1617〜1633.T）の直近1週間・1ヶ月の騰落率を集計し、
資金が流入しているセクター・流出しているセクターを Discord に通知します。

## 通知イメージ

```
📅 週刊【セクター別資金流向レポート】
直近1週間の業種別騰落率ランキング
----------------------------------
🔥 買われているセクター (TOP 5)
1. 情報・通信: +3.21% (月間: +5.1%)
2. 電機・精密: +2.85% (月間: +4.2%)
...
❄️ 売られているセクター (BOTTOM 5)
・銀行: -1.20% (月間: -0.8%)
...
```

## スケジュール

毎週土曜日 09:00 JST（`cron: '0 0 * * 6'`）

## Secrets

| 名前 | 内容 |
|---|---|
| `DISCORD_WEBHOOK` | Discord の Webhook URL |
